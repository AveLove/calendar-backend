## Installation

```
git clone https://github.com/dimaip/calendar-backend
cd calendar-backend
./install.sh
```

**Requires PHP 7.x+**

## Usage

Run `./start.sh` or put inside proper PHP web server.

```
GET /day.php?date=20180318
GET /bible.php?[translation=1Aver&]zachalo=%D0%91%D1%8B%D1%82.+XVII%2C+1-9.
```
